Start the Backend Server

The backend is the brain of your platform. It handles all data, users, and transactions.

What to do:

  1. Open a Terminal or Command Prompt
  2. Navigate to the cryptovault-api folder:
    cd cryptovault-api
  3. Start the server:
    npm start
  4. You should see: "Server running on port 5000"
  5. Keep this window open! Don't close it.
üí° Pro Tip

On Windows, you can double-click START.bat to start both servers automatically!

Start the Frontend (Website)

The frontend is what users see and interact with - the visual website.

What to do:

  1. Open a NEW Terminal window (keep the first one running!)
  2. Navigate to the app folder:
    cd app
  3. Start the frontend:
    npm run dev
  4. You should see: "Local: http://localhost:5173"
  5. Open your browser and go to: http://localhost:5173
‚ö†Ô∏è Important

You need BOTH servers running at the same time. You'll have 2 terminal windows open.

Manage Your Users

View all registered users, their balances, and account status.

What you can do:

  1. Click the "Users" tab in the Admin Panel
  2. See a list of all registered users
  3. View each user's:
    • Name and Email
    • Current Balance
    • Account Status (Active/Pending/Inactive)
  4. Click "View" on any user to see detailed information
  5. Use the Search box to find specific users
‚ö†Ô∏è User Actions

Approve or Reject Withdrawals

When users request to withdraw money, you need to approve it manually.

How to process withdrawals:

  1. Click the "Withdrawals" tab
  2. You'll see a list of pending withdrawal requests
  3. Each request shows:
    • User's name
    • Amount in USD
    • Amount in BTC
    • Bitcoin wallet address to send to
  4. To approve: Click the ‚úì (Approve) button
  5. To reject: Click the ‚úó (Reject) button
üí° Withdrawal Process

After you approve, you need to manually send the Bitcoin to the user's address from your own wallet. The system tracks the request but doesn't automatically send crypto.

Confirm User Deposits

When users send Bitcoin, you confirm it to credit their account.

How to confirm deposits:

  1. Click the "Deposits" tab
  2. See pending deposits waiting for confirmation
  3. Each deposit shows:
    • Amount in USD
    • Amount in BTC
    • Confirmation count (needs 3 to be valid)
  4. Verify you received the Bitcoin in your wallet
  5. Click ‚úì (Confirm) to credit the user's account
üí° Deposit Flow

1. User generates deposit address ‚Üí 2. User sends Bitcoin ‚Üí 3. You verify receipt ‚Üí 4. You confirm ‚Üí 5. User's balance is updated
